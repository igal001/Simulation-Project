void RPLapp::RPLapp_vDIOhandler ( DIOMessage *  msg) [protected, virtual]

Handles the incoming DIO messages and updates the DODAGtable accordingly – continues the DODAG construction. If the node decides to join the DODAG it saves/updates the new information regarding the DODAG in the DODAG table and sends a DIO message to its neighbors with the updated Rank and DTSN fields.
